# 3. その他仕様（要素に収まらない共通ルールや制約）

- カレンダーは月単位で切り替え可能
- ○×-は色分け（○=緑、×=赤、-＝グレー）
- 提出ボタンは必須日が1つ以上選択されていないと非活性
- レスポンシブ対応：スマホでは縦スクロール
- アクセシビリティ：カレンダー・ボタンはキーボード操作・コントラスト4.5:1以上
