# 3. その他仕様（要素に収まらない共通ルールや制約）

- **認証管理**: スタッフログイン必須、セッション管理
- **データ生成**: ダミーデータによる稼働情報の動的生成
- **ステータス管理**: 
  - 確定: 緑系（稼働場所・時間確定済み）
  - 現場未確定: 黄系（稼働予定だが場所未確定）
  - 休み: グレー系（稼働なし）
  - 詳細未確定: オレンジ系（稼働予定だが詳細未確定）
- **レスポンシブ対応**: 
  - デスクトップ: 2カラム表示（今日・明日並列）
  - タブレット: 2カラム表示（縮小版）
  - スマホ: 1カラム表示（縦並び）
- **アクセシビリティ**: 
  - WCAG 2.2 AA 準拠
  - キーボード操作完全対応
  - コントラスト4.5:1以上
  - スクリーンリーダー対応
- **UI/UX**: 
  - 稼働詳細ドロワーは右スライドイン形式、幅900px
  - カードホバー効果・アニメーション
  - ローディング状態表示
  - エラー状態のフォールバック表示
- **データ表示**: 
  - 稼働時間: 平日11:00-19:00、土日10:00-18:00
  - 日当: 平日¥15,000、土日¥18,000
  - 集合時間: 稼働開始30分前
  - 代理店: ANSTEYPE、イベントプロ、マーケティングラボ等
- **コミュニケーション機能**: 
  - 管理者とのリアルタイムメッセージ
  - 既読・未読状態管理
  - ファイル添付未対応
- **パフォーマンス**: 
  - 軽量なダミーデータ生成
  - 効率的な状態管理
  - 最小限のAPI通信
- **セキュリティ**: 
  - スタッフ認証必須
  - 他スタッフ情報の非表示
  - セッションタイムアウト対応
- **注意事項**: 
  - 現在はダミーデータによる表示
  - 実際のAPI連携は未実装
  - リアルタイム更新機能は未実装
